# Python TOTP Microservice

Этот проект представляет собой микросервис для генерации и проверки одноразовых кодов (TOTP) на Python.  
Он переписан с оригинального Google Authenticator для Android, который написан на Java и Objective-C.  
Оригинальный проект Google Authenticator доступен здесь: [https://github.com/google/google-authenticator](https://github.com/google/google-authenticator).  
Цель переписывания — сделать компактный, полностью рабочий Python микросервис для обучения и интеграции.  
Проект использует FastAPI для создания REST API, что позволяет легко интегрировать его с веб-приложениями.  
Сервис поддерживает генерацию TOTP кодов по времени с шагом 30 секунд.  
Также реализована проверка введённых кодов с допуском ±1 time step, как в Google Authenticator.  
Для генерации TOTP используется стандартный алгоритм HMAC-SHA1 и Base32-секреты.  
Проект включает функции для генерации QR-кодов, чтобы можно было подключать их к мобильным аутентификаторам.  
Модель пользователя описана с помощью Pydantic, что упрощает хранение и валидацию данных.  
Все зависимости указаны в `requirements.txt` и легко устанавливаются через pip.  
Структура проекта минимальная: 5 файлов для быстрого старта и понимания логики.  
Этот проект можно использовать как основу для своих сервисов двухфакторной аутентификации или как учебный пример TOTP на Python.
